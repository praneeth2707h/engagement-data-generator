## Generated Artifacts — Delete Before Upload

These are automatically regenerated by Python and pytest. They do not belong in a repository.

| File / Folder | Classification | Reason |
|---|---|---|
| `__pycache__/` (all occurrences) | ❌ **DELETE** | Python bytecode cache — auto-generated |
| `*.pyc` | ❌ **DELETE** | Compiled Python files — auto-generated |
| `.pytest_cache/` (all occurrences) | ❌ **DELETE** | pytest run cache — auto-generated |
| `.coverage` | ❌ **DELETE** | Coverage data — auto-generated |
| `.coverage.*` | ❌ **DELETE** | Coverage partial files — auto-generated |

**How to delete these on a Mac:**
Open Terminal in the project folder and run:
```bash
find . -type d -name __pycache__ -exec rm -rf {} + 2>/dev/null
find . -name "*.pyc" -delete
rm -rf .pytest_cache tests/.pytest_cache
rm -f .coverage .coverage.*
```

**How to delete these on Windows:**
Open Command Prompt in the project folder and run:
```
for /d /r . %d in (__pycache__) do @if exist "%d" rd /s/q "%d"
del /s /q *.pyc
rmdir /s /q .pytest_cache
del .coverage
```
